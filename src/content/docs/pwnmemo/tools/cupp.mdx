---
title: cupp
description: How to use the tool "cupp" to generate custom wordlists based on a persons data points.
---

```shell
cupp -i
```

Run the above command and navigate through the wizard, entering every information on your target you have. After
completing the wizard, `cupp` will write a list of possible usernames and passwords to a text file.
